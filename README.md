# PlanIt: Multi-Agent AI Planner

PlanIt is a distributed multi-agent architecture designed for intelligent trip and logistics planning. The system demonstrates how specialized agents collaboratively optimize route, cost, time, and resource constraints.

## Features

- Multi-agent modular architecture
- Preference-driven optimization
- Decision fusion mechanism
- Python simulation prototype

## Project Structure

- planit_simulation.py — main simulation code
- architecture_diagram.png — system architecture

## How to Run

```bash
python planit_simulation.py
